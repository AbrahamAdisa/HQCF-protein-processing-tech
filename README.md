# High Quality Cassava Flour with Protein Technology

This project focuses on developing protein-enriched cassava flour (HQCF+) to improve nutritional value and expand its use in food production, particularly in Africa. Also, underutilization and post harvest contributing to low value casava commercialization.

## Problem Statement:
Cassava is a staple crop that is high in carbohydrates but low in protein. Protein deficiency is prevalent in cassava-consuming communities, leading to malnutrition.

## Our Solution:
We are developing an innovative processing technique to fortify HQCF with plant-based proteins (e.g., soybean or cowpea flour) to create a balanced, nutritious product,by adoption of processing technolLink

## Project Objectives:
- Produce HQCF from cassava roots.
- Fortify HQCF with plant-based protein.
- Test for nutritional content and functionality.
